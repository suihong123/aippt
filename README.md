# AIPPT生成器

基于AI的智能PPT生成工具。

## 功能特点

- 智能生成PPT
- 用户管理系统
- 激活码系统
- 性能监控
- 数据备份

## 在线使用

访问 [https://yourusername.github.io/aippt/](https://yourusername.github.io/aippt/)

## 本地开发

1. 克隆仓库
```bash
git clone https://github.com/yourusername/aippt.git
cd aippt
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行开发服务器
```bash
python app.py
```

## 部署

项目使用 GitHub Pages 部署，推送到 main 分支后会自动部署。
